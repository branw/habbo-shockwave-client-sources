## Generator

This repository was generated using the `main.py` script in this branch. It
takes a folder of releases as input and generates the `sources` branch.
Decompilation is performed using 
[ProjectorRays](https://github.com/ProjectorRays/ProjectorRays).

### Usage

1. Create a `releases` folder in the working directory and copy over all
   input releases, separated into individual folders.
2. Update the `RELEASE_ORDER` list in `main.py` with any new releases that
   are not already present.
3. [Download the latest `projectorrays.exe`](https://github.com/ProjectorRays/ProjectorRays/releases/tag/latest)
   into the working directory.
4. Install `GitPython` from PyPi, e.g. `pip install GitPython`.
5. Execute `main.py`, e.g. `py main.py`.

A fresh Git repository is then created in `staging/`; any existing content will
be deleted.
